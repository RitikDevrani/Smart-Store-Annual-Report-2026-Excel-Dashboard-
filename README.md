# Smart Store Annual Report 2026 (Excel Dashboard)
## Project Overview

This project presents an interactive Excel dashboard for analyzing annual sales performance of a smart store in 2026.

The dashboard provides insights into sales trends, customer demographics, order status, regional performance, and sales channels, helping businesses make data-driven decisions.

## Objectives

The main objectives of this project are:

• Analyze yearly sales performance

• Compare order count vs sales amount

• Understand customer demographics (age & gender)

• Identify top-performing states

• Analyze order status (Delivered, Cancelled, Returned)

• Evaluate sales across different channels

## Dashboard Features

The dashboard includes the following visualizations:

• Order vs Sales Trend (Monthly)

• Sales Distribution (Men vs Women)

• Order Status Analysis

• Top 5 States by Sales

• Age Group vs Gender Analysis

• Sales by Channels (Amazon, Flipkart, Myntra, etc.)

## Filters / Slicers

Interactive filters are provided for better analysis:

• Month filter

• Channel filter

• Category filter

These filters allow users to dynamically explore the data.

## Key Insights

Some important insights from the dashboard:

• Women contribute more to total sales compared to men

• Majority of orders are successfully delivered (~90%+)

• Amazon and Myntra are the top-performing sales channels

• Maharashtra, Karnataka, and Uttar Pradesh generate highest sales

• Adult age group contributes the most to total orders
